# Bidding App

### Backend Stack

- NodeJS, TypeScript
- Express
- Lodash
- WebSocket

## Installation

Install the dependencies and devDependencies and start the server.

```sh
cd server
npm install
./node_modules/.bin/tsc
node ./dist/server/server.js
```

### Mobile Stack

- Flutter V 2.10.4
- dio
- web_socket_channel
- jiffy (similarly momentjs)

## Installation

```sh
cd mobile
fvm flutter pub get
fvm flutter run
```
